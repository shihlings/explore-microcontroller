Bi-weekly Reflection - November 26th, 2021

November 26th, 2021

Over the past weeks, I've been mainly thinking about what I should do for a final product. However, I got stuck with coming up with an idea that can fully involve all devices on the board.

I started with the hardware check program first. The hardware check program runs through all devices and provides a quick way to check if anything is malfunctioning. The code is still messy, but it is usable. Most of the code came from previous device experiments, as the device experiments were to make sure that the device worked as expected. I created multiple functions in the code and called the functions in the main function.

I was planning on doing one final product. However, I am moving towards multiple small products as my final product. In the following weeks, I will be working on more products and fixing the mess code I have in my previous projects. I will also be working on my circuit diagram as it is overdue.

I am starting to procrastinate, and my deadlines are quickly approaching. I will try to make more time and work on stuff that lacks progress.
