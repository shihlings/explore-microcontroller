Buzzer

Introduction

The Buzzer is a General-Purpose Input/Output (GPIO) device, and it is connected to the Raspberry Pi Pico directly through GPIO/PIO ports. The black wire connects to GND, and the red wire connects to any GPIO PIN. The Buzzer only has the function to turn on and off and cannot adjust its frequency.

In this test, the program turns on the Buzzer and turns it off, each time with a different time interval. It can be used in different case scenarios, for example, errors, signal receiving indicators, etc.

Demonstration

The program is initiated by the Raspberry Pi 4B (the touch screen in the upper part of the screen), and the Raspberry Pi Pico runs the program.

The program turns on Buzzer for one second. Then, it turns off the Buzzer and sleeps for one second. The program will repeat another three times, but with the Buzzer on for 0.5, 0.2, and 0.1 seconds.
