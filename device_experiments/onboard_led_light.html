Onboard LED Light (GPIO 25)

Introduction

The Onboard LED Light is a General-Purpose Input/Output (GPIO) device. It is built on the PCB of the Raspberry Pi Pico and is assigned its GPIO PIN (PIN 25).

In this test, the program is designed to turn on and off the Onboard LED Light. The Onboard LED Light can be used for power indication or other purposes. The drawback of this LED Light is that it is irreplaceable, and no other color option is available.

Demonstration

The program is initiated by the Raspberry Pi 4B (the touch screen in the upper part of the screen), and the Raspberry Pi Pico runs the program.

The program turns on Onboard LED Light and prints "On board LED on" on the Raspberry Pi 4B. Then, it turns off the Onboard LED Light and prints "On board LED off" on the Raspberry Pi 4B.
