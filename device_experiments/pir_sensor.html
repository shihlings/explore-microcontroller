Passive Infrared Sensor (PIR Sensor)

Introduction

The Passive Infrared Sensor (PIR Sensor) used in this test is an HC-SR501 PIR Sensor. Device specifications can be found here or by clicking the button below.

In this test, the program prints messages on the Raspberry Pi Pico when the PIR Sensor detects movement. The PIR Sensor is programmed as an interrupt so that it interrupts the program and executes the code whenever it detects something. The PIR Sensor is useful in real-life cases such as automatic staircase lights when someone approaches, burglar alarm to notify owners when there is movement when not authenticated, etc.

Demonstration

The program is initiated by the Raspberry Pi 4B (the touch screen in the upper part of the screen), and the Raspberry Pi Pico runs the program.

The program initiates the PIR sensor upon running, and it will print "Motion detected" on the Raspberry Pi 4B when movement is detected. Then, it will timeout for five seconds to avoid repeated signals. Afterward, it prints "5s..." on the Raspberry Pi 4B when the detection resumes. The program is inside an always true while loop, so it will only exist when the user force stops the program.
