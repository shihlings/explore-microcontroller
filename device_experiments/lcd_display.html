LCD Display

Introduction

The LCD Display used in this test is an I2C 1602 LCD Display. It is a 16 x 2 LCD Display, and it uses the I2C protocol to communicate with the Raspberry Pi Pico.

In this test, two additional APIs from GitHub (https://github.com/T-622/RPI-PICO-I2C-LCD) were added to enable functions such as: lcd.putstr(), lcd.clear(), lcd.blink_cursor_on(), etc. The main purpose of this test enables the LCD Display to display its I2C address and "Hello, World!", turn on or off its backlight, and turn on, off, or hide the cursor. With the LCD Display set up, it can be used to replace the print() function, which prints to the computer, and display instantaneous messages without a computer connected.

Demonstration

The program is initiated by the Raspberry Pi 4B (the touch screen in the upper part of the screen), and the Raspberry Pi Pico runs the program.

The program displays the LCD Display's I2C address and the message "Hello, World!" to test the LCD Display's ability to print messages. Then, the program turns the LCD Display's backlight on and off repeatedly. Finally, the program prints the numbers 0 to 9 on the display and exits.

The LCD Display issue was addressed during development and has been fixed at the time of recording. Details can be found by clicking here or the button below. No additional issues were encountered.
