Buttons

Introduction

The Button is a General-Purpose Input/Output (GPIO) device, and it is connected to the Raspberry Pi Pico directly through GPIO/PIO ports. Its legs on the top are interconnected, and the same for the bottom ones. The Button passes the current from one side to the other when it is pushed. Therefore, it can be set up by simply connecting one side to the VCC and the other to a GPIO PIN.

In this test, the program detects if the Button is pressed. It prints a message when the buttons are pressed.

Demonstration

The program is initiated by the Raspberry Pi 4B (the touch screen in the upper part of the screen), and the Raspberry Pi Pico runs the program.

The program prints "Button 1 pressed" when the button on the right is pressed, and prints "Button 2 pressed" when the button on the left is pressed.

The Signal Bouncing issue was addressed during development and has been fixed at the time of recording. Details can be found by clicking here or the button below. No additional issues were encountered.
