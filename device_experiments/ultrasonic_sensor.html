Ultrasonic Sensor

Introduction

The Ultrasonic Sensor used in this test is an HC-SR04 Ultrasonic Sensor. Device specifications can be found here or by clicking the button below.

In this test, the program prints the distance measured by the Ultrasonic Sensor on the Raspberry Pi Pico. The Ultrasonic Sensor can act as eyes when it is installed on a robot and can detect obstacles easily.

Open Device Datasheet


Demonstration

The program is initiated by the Raspberry Pi 4B (the touch screen in the upper part of the screen), and the Raspberry Pi Pico runs the program.

The program initiates the Ultrasonic Sensor upon running, and it will print the distance on the Raspberry Pi 4B continuously.
